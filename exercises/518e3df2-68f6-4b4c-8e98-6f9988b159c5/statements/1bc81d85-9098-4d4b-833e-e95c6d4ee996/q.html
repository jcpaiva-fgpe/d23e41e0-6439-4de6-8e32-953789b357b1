Funkcja obliczająca punkty w grze w kości.

Na wejściu otrzymuje rzut w postaci pięciu liczb oznaczających liczbę oczek wyrzuconych na każdej kości. Na wyjściu zwraca słownik, w którym dla każdej punktowanej kategorii zwraca liczbę punktów uzyskanych w rzucie: - jedynki: suma wyrzuconych jedynek (lub 0 jeśli nie wyrzucono żadnej jedynki), - dwójki: suma wyrzuconych dwójek (lub 0 jeśli nie wyrzucono żadnej dwójki), - trójki: suma wyrzuconych trójek (lub 0 jeśli nie wyrzucono żadnej trójki), - czwórki: suma wyrzuconych czwórek (lub 0 jeśli nie wyrzucono żadnej czwórki), - piątki: suma wyrzuconych piątek (lub 0 jeśli nie wyrzucono żadnej piątki), - szóstki: suma wyrzuconych szóstek (lub 0 jeśli nie wyrzucono żadnej szóstki), - trzy jednakowe: suma oczek na wszystkich kościach jeśli wyrzucono trzy kości z taką samą liczbą oczek (lub 0 jeśli nie wyrzucono), - cztery jednakowe: suma oczek na wszystkich kościach jeśli wyrzucono cztery kości z taką samą liczbą oczek (lub 0 jeśli nie wyrzucono), - full: 25 punktów jeśli wyrzucono trzy kości z jednakową liczbą oczek i pozostałe dwie kości z jednakową liczbą oczek (lub 0 jeśli nie wyrzucono), - mały strit: 30 punktów jeśli wyrzucono cztery kości o liczbie oczek rosnącej o jeden (lub 0 jeśli nie wyrzucono), - duży strit: 40 punktów jeśli wyrzucono pięć kości o liczbie oczek rosnącej o jeden (lub 0 jeśli nie wyrzucono), - generał: 50 punktów jeśli wyrzucono pięć kości o takiej samej liczbie oczek (lub 0 jeśli nie wyrzucono), - szansa: suma oczek na wszystkich kościach.
